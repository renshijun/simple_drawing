This simple drawing program can be used to create figures that can be inserted to academic papers or ppts. The program has no 
graphic interface. Key sequence is used to tell the program what to do. For example, if you want to draw a line, just type "d" "l"
then left click the starting point, move the pointer to the ending point and right click.

All key sequence don't need to remember, use help to look at the key sequence.
